3. 模組與套件

(1) 召喚魔法

一個模組(module)就是一個.py檔，需要時就被使用者召喚出來

- 那為什麼Python不把所有技能集於一身，危急關頭還要靠咒語召喚???

(2) 模組種類多元

例如在處理時間的模組有time,datatime
處理資料的有numpy,pandas
爬蟲的有requests,selenium,BeautifulSoup

(3) 召喚組合技

首先要先將模組載入電腦

pip install time 
####pip是python的套件管理員####
conda install time 
####Anaconda是Python與R的一種簡化軟體套件管理工具
####我們這次用的就是Anaconda，需要時前面要叫一聲「conda~~~~」

需要時就將他們import進來

from selenium import webdriver #只從selenium載入webdriver
from bs4 import BeautifulSoup
import pandas as pd #簡化名稱
from pandas import * #載入模組中全部功能

然後怎麼用呢!!!通常是主咒語加上次要咒語
譬如說當我們import pandas as pd之後，召喚方式就是pd.[某個功能]，例如我們如果要建立Dataframe，可以用下列方式。

pet = {
    "name": ["dog", "rabbit", "lion", "cat"],
    "weight": [80, 75, 93, 86],
    "height": [63, 90, 85, 70]
}
 
df = pd.DataFrame(pet)

小撇步:如果對這些套件有疑義，建議可以去看套件開發的原始文件唷~例如:
https://pandas.pydata.org/docs/reference/api/pandas.DataFrame.html

解答

for i in range(len(C)):
    #print(i)
    print(C["rabbit"][i])

第一層爬蟲

  • STEP1:檢視網頁結構


(圖片來源:https://medium.com/@troy801125/web-介紹-47c667043d72)

  • STEP2:運用套件把網頁html抓下來

可能遭遇的困難:
(1) 網頁標籤沒有規律
(2) 遇到需要登入或有網頁偏好設定才能訪問的網頁(解決方式:需要帶著cookie)
(3) 網頁會偵測到有人大量爬取資料而封鎖對方(解決方式:切換IP位址、設定Request Headers、設定User-Agent、
設定延遲時間、使用Headless Browser)

  • STEP3:以迴圈處理
  • STEP4:存放入儲存格
cookies = {"over18" : "1"} 
res = requests.get(
    "https://www.ptt.cc/bbs/Gossiping/index.html", cookies=cookies)
mainPageDoc = pq(res.text)
dataList = mainPageDoc("#main-container div > div.title > a")

for each in dataList.items():
    K["title"].append(each.text()) 
    cateRes = "https://www.ptt.cc{}".format(each.attr("href"))
    K["url"].append(cateRes)

K = pd.DataFrame(K)

第二層爬蟲

for i in range(len(K["url"])):
  cookies = {"over18" : "1"}
  res = requests.get(
    K["url"][i], cookies=cookies)
  soup = BeautifulSoup(res.text, 'html.parser')
  for a in soup.find_all(attrs={"id": "main-container"}):
    textP = a.text
    textP2 = textP.split("--")[0]
    textP3 = textP2.split("\n")
    DF["title"].append(K["title"][i])
    DF["url"].append(K["url"][i])
    DF["content"].append(textP3)

DF= pd.DataFrame(DF)

DF["content"][1]

儲存檔案

DF.to_csv("PPT八卦版test.csv")

進入頁面

url = "https://m.facebook.com/udnplus?locale=zh_TW"  

driver = webdriver.Chrome("C:\Program Files (x86)\Google\Chrome\Application\chromedriver.exe") #打開虛擬瀏覽器
driver.get(url) #取得連結

讓動態網頁往下滑，才能載入網頁內容。然後使用bs4套件載入網頁html


#下滑頁面
for x in range(3):
    driver.execute_script("window.scrollTo(0,document.body.scrollHeight)")
    print("scroll")
    time.sleep(5)
    
#讓bS4載入頁面
soup = BeautifulSoup(driver.page_source)

建立儲存格

post = {
    "title": [],
    "url": []
}
DF = {
    "title": [],
    "url": [],
    "name": [],
    "content": [],
    "time": []
}

爬取貼文的標題與連結

for a in soup.find_all(attrs={"class": "_5rgt _5nk5 _5msi"}):
    post["title"].append(a.text)
    #print(a.get("href"))
for a in soup.find_all("a", {"class": "_5msj"}):
    post["url"].append(a.get("href"))

轉換連結格式

for q in range(len(post["url"])):
    post["url"][q] = "https://m.facebook.com/"+post["url"][q]

進入每一條連結，爬下留言

for w in range(len(post["url"])): #大迴圈:網址
    driver.get(post["url"][w])
    time.sleep(5)
    soupNext = BeautifulSoup(driver.page_source, "lxml") #爬到網址的html
    
    #檢查是不是影片
    if re.search("Facebook Watch", str(soupNext)):
        print("影片")
    else:
        print("非影片")
        comment_Name = soupNext.find_all(class_="_2b05")  # 留言的人
        comment_Content = soupNext.find_all(
        attrs={"data-sigil": "comment-body"})  # 留言的內容
        comment_Time = soupNext.find_all(class_="_4ghv _2b0a")  # 留言的時間
        for i in comment_Name:
            print(i.text)
            DF["title"].append(post["title"][w])
            DF["url"].append(post["url"][w])
            DF["name"].append(i.text)
        for y in comment_Content:
            print(y.text)
            DF["content"].append(y.text)
        for o in comment_Time:
            print(o.text)
            DF["time"].append(o.text)
            
    time.sleep(5)

把剛剛存成dict的檔案轉成dataframe

DF = pd.DataFrame(DF)
DF

儲存檔案

DF.to_csv("FB聯合報留言test.csv")
