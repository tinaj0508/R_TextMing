&(與，全都要符合)、|(或，符合其一就好)、!(排除)

在此順便介紹「正則表示式」【非常重要】 > 規則可參考:https://blog.yjtseng.info/post/regexpr/

ClassA <- c("May","Sam","Kate","Adam")
url = "https://blog.yjtseng.info/post"

newurl <- gsub("blog.yjtseng.info/post","習近平",url)

練習1

假設您正在開發一個網頁應用程式，需要從一個文本框中提取日期。請編寫一個正則表達式，從文本中提取日期，日期格式為YYYY-MM-DD。 請跟chatgpt要答案，並按照他的答案實作一遍。 hint:如果chatgpt只給你正則表達式的代碼，請追問如何提取日期 如果chatgpt用了sub()，把日期替換掉，請糾正他。

text <- "發文時間:2022-02-04
純欲系女孩 啾啾
👑 身高/體重：160/40
👑 血型：A
👑 生日/星座：5/26 雙子
👑 三圍：30E/21/33
💞 興趣：賺錢工作 睡覺 保養 拍照
💞 專長：工作賺錢 拍照 打扮自己 搭車
🖤 最怕：蚯蚓 沒錢
關閉留言時間:2022-04-05"

# 使用 gsub 函數替換非數字和非日期字符
text <- gsub("[^0-9-]", "", text)

# 匹配日期格式的正則表達式
date_regex <- "\\d{4}-\\d{2}-\\d{2}"

# 提取匹配的日期格式
date_matches <- regmatches(text, gregexpr(date_regex, text))

# 將匹配結果轉換為日期格式
dates <- as.Date(date_matches[[1]])

#這個代碼會首先使用gsub函數替換非數字和非日期字符，將文本中的日期格式分離出來。然後，它使用與之前相同的正則表達式和regmatches函數，從文本中提取出匹配的日期格式。最後，它使用as.Date函數將匹配的日期字符串轉換為日期格式。
